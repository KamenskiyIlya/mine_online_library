# Собственная онлайн библиотека

<img width="64" height="64" alt="logo" src="https://github.com/user-attachments/assets/8ac78018-8e85-44c3-afdc-9e88c82c64c4" />

Сайт представляет из себя онлайн библиотеку с книгами. На сайте отображаются обложки книг, их названия, автор и жанры, в которых они написаны. При этом нужную книгу можно читать прям в браузере.

<img width="1491" height="845" alt="image" src="https://github.com/user-attachments/assets/8aa5469c-c4a6-4491-b82d-6e2dcaa5e0b1" />

### Особенность

Скрипт генерирует HTML страницы из json файла `meta_data.json`, где содержится вся информация о книгах. Делает он это с помощью шаблонизатора `Jinja2`. Скриптом автоматически формируются отдельные страницы с определенным кол-вом отображаемых на них книг, по страницам можно переключаться с помощью панели навигации, которая расположена вверху страниц.

<img width="713" height="121" alt="image" src="https://github.com/user-attachments/assets/cc7963d3-1319-4ece-987e-9546c393697a" />

Если нажать на кнопку "Читать", тогда откроется новая вкладка с содержимым выбранной книги.

После запуска скрипта данной библиотекой можно будет пользоваться и оффлайн, для этого через браузер откройте любую страницу из папки `pages`. Оффлайн версия будет также функционировать как и запущенный сайт.

### Пример сайта

Пример того как будет выглядеть сайт можете посмотреть перейдя по [ссылке](https://kamenskiyilya.github.io/mine_online_library/pages/index1.html).

### Что потребуется

Для запуска локального сервера вам понадобится Python третьей версии.

Скачайте код с GitHub. Затем установите зависимости

```
pip install -r requirements.txt
```

После этого можно запускать локальный сервер. Для генерации страниц с помощью `livereload` используйте команду:
```
python render_website.py
```

Открыть отрисованный сайт можно по ссылке:
```
http://127.0.0.1:8000
```

### Продвинутые функции

При запуске скрипта у Вас есть возможность задать свои свойства. Сделать это Вы можете указав имя параметра, который хотите поменять и значение параметра в следующем формате:
```
python render_website.py -bop 20
```
Доступные для вызова параметры:

`-h` - флаг, который выведет подсказки о всех доступных флагах, их описание и прочую полезную информацию.
`-tp` - путь и название файла шаблона для отрисовки страниц. Если Вы хотите отрисовывать сайт с помощью другого шаблона или шаблон находится в другой директории и в директорию к скрипту Вы его не хотите перемещать, можно просто указать путь и имя нужного файла вместе с расширением.
`-bf` - путь и название файла, который содержит данные о книгах, указывать необходимо вместе с расширением файла.
`-bop` - Вы можете задать кол-во книг отображаемых на одной странице, по умолчанию это 10 книг.
