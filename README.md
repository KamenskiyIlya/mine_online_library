# Собственная онлайн библиотека

<img width="64" height="64" alt="logo" src="https://github.com/user-attachments/assets/8ac78018-8e85-44c3-afdc-9e88c82c64c4" />

Сайт представляет из себя онлайн библиотеку с книгами. На сайте отображаются обложки книг, их названия, автор и жанры, в которых они написаны.

<img width="1491" height="845" alt="image" src="https://github.com/user-attachments/assets/8aa5469c-c4a6-4491-b82d-6e2dcaa5e0b1" />

### Особенность

Скрипт генерирует HTML страницы из json файла `meta_data.json`, где содержится вся информация о книгах. Делает он это с помощью шаблонизатора `Jinja2`. Скриптом автоматически формируются отдельные страницы с определенным кол-вом отображаемых на них книг, по страницам можно переключаться с помощью панели навигации, которая расположена вверху страниц.

<img width="713" height="121" alt="image" src="https://github.com/user-attachments/assets/cc7963d3-1319-4ece-987e-9546c393697a" />

Если нажать на кнопку "Читать", тогда откроется новая вкладка с содержимым выбранной книги.

### Пример сайта

Пример того как будет выглядеть сайт можете посмотреть перейдя по [ссылке](https://kamenskiyilya.github.io/mine_online_library/pages/index1.html).

### Что потребуется

Для запуска сайта вам понадобится Python третьей версии.

Скачайте код с GitHub. Затем установите зависимости

```
pip install -r requirements.txt
```

После этого можно запускать сайт. Для генерации страниц с помощью `livereload` используйте команду:
```
python render_website.py
```

Открыть отрисованный сайт можно по ссылке:
```
http://127.0.0.1:8000
```
